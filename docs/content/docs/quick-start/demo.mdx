---
title: Run the interactive demo
description: Use the bundled React demo against Phoenix or the mock transport.
---

The repository ships an interactive demo under `example/javascript_demo`.

```bash
cd example/javascript_demo/interactive
npm ci
npm run dev
```

Follow the additional notes in `example/javascript_demo/README.md` — the demo expects a running Phoenix instance when you are not using the mock transport.

<Callout type="info" title="Documentation site">
  This `docs/` Next.js app is only the documentation shell. It does not replace the interactive demo for experimenting with UI flows.
</Callout>
