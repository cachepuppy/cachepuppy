title: Quick start
description: Run CachePuppy with Docker, then use the JavaScript client in a few lines of code.
---

You will:

1. [Run the server with Docker](./run-with-docker) — one command brings up the clustered Elixir backend behind nginx.
2. [Use JavaScript](./javascript) — connect, subscribe, publish, and read/write cache keys.

That is enough to feel end‑to‑end latency and cache behavior locally. When you need the full mental model, continue to [Core concepts](/docs/core-concepts) and the [JavaScript](/docs/javascript) chapter.
