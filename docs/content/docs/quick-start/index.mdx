title: Quick start
description: From zero to a running backend, built SDKs, and a minimal realtime snippet.
---

This section gets you productive quickly. You will:

1. Confirm prerequisites on your machine.
2. Run the Phoenix application.
3. Build the JavaScript core and React SDK packages.
4. Open a websocket, subscribe to a topic, and publish an event.
5. Optionally run the interactive demo app in `example/javascript_demo`.

Use the sidebar pages in order, or jump straight to [Hello realtime](./hello-realtime) if your backend is already running.
