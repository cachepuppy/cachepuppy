---
title: JavaScript SDK (@cachepuppy/core)
description: Connection-oriented client, wire protocol helpers, transports, and the admin HTTP client entry point.
---

The **`@cachepuppy/core`** package is the foundation for browser and Node integrations. It exposes:

- **`createClient`** / **`CachePuppyClient`** — websocket-first workflows with Phoenix or an in-memory mock.
- **`createAdminClient`** / **`CachePuppyAdminClient`** — HTTP-only workflows documented in the [Admin HTTP client](/docs/admin-http-client) section.
- **Protocol helpers** — `createEnvelope`, `isEnvelope`, `nextId`, and the **`PhoenixTransport`** implementation.
