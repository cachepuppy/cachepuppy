CachePuppy is a **topic-first realtime layer** with optional **distributed cache** access. This documentation tracks the implementations in this monorepo:

- **Elixir / Phoenix** (`cachepuppy_core`) — websockets, topic processes, HTTP routes.
- **`@cachepuppy/core`** — typed client, wire envelopes, Phoenix + mock transports, admin HTTP client.
- **`@cachepuppy/react`** — provider and hooks built on the core client.
