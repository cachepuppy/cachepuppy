# javascript_demo

Single Next.js app that demonstrates CachePuppy's three primary capabilities —
caching, realtime cursors, and workflows — under one roof. See
[`unified/README.md`](./unified/README.md) for setup and run instructions.

From the repo root the easiest way to launch it is:

```bash
make demo-unified
```

That target builds both SDKs (`@cachepuppy/core`, `@cachepuppy/react`),
installs the demo's dependencies, and starts Next.js on
<http://localhost:3000>.
